Connect to App Store
Enter your Apple Developer API credentials once. POSTAPP handles everything else.
1
Go to appstoreconnect.apple.com and sign in with your Apple Developer account.
2
Tap Users and Access in the top menu, then choose Integrations → App Store Connect API.
3
Click Generate API Key. Give it any name and set the role to Admin.
4
Copy the Issuer ID shown at the top of the page, and the Key ID shown next to your new key.
5
Click Download API Key to get the .p8 file. Open it in any text editor and paste the full contents into the Private Key field below.
6
Codemagic Token (optional): Log in to codemagic.io, go to Teams → Integrations → Codemagic API, and copy your token. Only required to trigger builds directly from this app.
